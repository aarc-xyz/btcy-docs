## Overview

iBTCY is an ERC-20 token with transfer restrictions for regulatory compliance.

---

## Contract Details

| Property | Value |
|----------|-------|
| **Standard** | ERC-20 |
| **Upgradeable** | Yes (UUPS) |
| **Pausable** | Yes |
| **Access Control** | Role-based |

---

## Key Functions

### Transfers

```solidity
function transfer(address to, uint256 amount) external returns (bool);
function transferFrom(address from, address to, uint256 amount) external returns (bool);
```

Transfers are restricted:
- Recipient must be on allowlist
- Sender must not be on blocklist
- Contract must not be paused

### Allowlist

```solidity
function isAllowed(address account) external view returns (bool);
```

Check if an address can hold iBTCY.

### Blocklist

```solidity
function isBlocked(address account) external view returns (bool);
```

Check if an address is blocked.

## Events

```solidity
event Transfer(address indexed from, address indexed to, uint256 value);
event Approval(address indexed owner, address indexed spender, uint256 value);
event AllowlistUpdated(address indexed account, bool allowed);
event BlocklistUpdated(address indexed account, bool blocked);
event Paused(address account);
event Unpaused(address account);
```

## Errors

```solidity
error RecipientNotAllowed(address recipient);
error SenderBlocked(address sender);
error ContractPaused();
```

## Access Control

| Role | Functions |
|------|-----------|
| `COMPLIANCE_ROLE` | `setAllowed`, `setBlocked` |
| `PAUSER_ROLE` | `pause`, `unpause` |
| `UPGRADER_ROLE` | `upgradeTo` |
