# iBTCY Token

iBTCY (Institutional BTCY) is a **regulated security token** representing ownership in a Swiss Actively Managed Certificate (AMC) that generates yield on Bitcoin.

## Key Properties

| Property | Value |
|----------|-------|
| **Standard** | ERC-20 + ERC-2612 Permit |
| **Decimals** | 8 (matches Bitcoin) |
| **Transfer Rules** | Allowlist + Blocklist |
| **Pricing** | NAV-based (grows ~5% APY) |

## NAV-Based Pricing

The Net Asset Value (NAV) represents the value of 1 iBTCY in BTC:

- **Starting NAV**: 1.0 (1 iBTCY = 1 BTC)
- **Growth**: ~5% annually from yield strategies
- **Example**: At NAV 1.05, 100 BTC mints ~95.24 iBTCY

<Info>
  NAV is updated via Chainlink oracle with multiple safety checks to prevent manipulation.
</Info>

## Transfer Restrictions

iBTCY uses dual compliance controls:

### Blocklist (Checked First)
- Prevents sanctioned/flagged addresses from transacting
- Applied to sender AND receiver
- Fail-fast pattern for gas efficiency

### Allowlist
- Only KYC'd addresses can hold iBTCY
- Required for sender, receiver, AND spender (in transferFrom)
- Can be globally disabled by admin if needed

## Force Capabilities

For compliance/legal requirements, authorized roles can:

| Function | Requirement | Use Case |
|----------|-------------|----------|
| `forceTransfer()` | User must be blocklisted first | Court orders, estate claims |
| `forceRedeem()` | User must be blocklisted first | Sanctions compliance |

<Warning>
  Force actions require the target address to be blocklisted first, creating a clear audit trail.
</Warning>
