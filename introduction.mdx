## What is BTCY?

BTCY is an institutional-grade Bitcoin yield product designed for DeFi composability. It bridges **regulated finance** with **permissionless protocols** through a two-token architecture.

<CardGroup cols={2}>
  <Card title="iBTCY" icon="building-columns" color="#A5E547">
    **Institutional Token**
    
    NAV-based security token backed by Sygnum's actively managed BTC yield fund. Restricted to qualified investors.
  </Card>
  <Card title="BTCY" icon="coins" color="#A5E547">
    **DeFi Wrapper**
    
    ERC-4626 vault token. 1:1 backed by iBTCY. Fully composable with lending, DEXs, and collateral protocols.
  </Card>
</CardGroup>

## Why BTCY?

<AccordionGroup>
  <Accordion title="Native Bitcoin Yield" icon="bitcoin" defaultOpen>
    Bitcoin sitting idle doesn't earn yield. BTCY unlocks ~5% APY through Sygnum's institutional-grade BTC yield strategies—without selling your BTC exposure.
  </Accordion>
  <Accordion title="DeFi Composability" icon="puzzle-piece">
    Most institutional products can't enter DeFi. BTCY's two-token design separates compliance (iBTCY) from composability (BTCY), letting you use your yield-bearing BTC anywhere.
  </Accordion>
  <Accordion title="Regulatory Clarity" icon="shield-check">
    Swiss AMC structure with BMA-licensed custody at Sygnum Bank. Clear legal framework, not regulatory arbitrage.
  </Accordion>
  <Accordion title="Institutional Custody" icon="vault">
    Assets held by Sygnum Bank (Switzerland) with Archax as execution partner. Bank-grade security for your Bitcoin.
  </Accordion>
</AccordionGroup>

## How It Works

```
┌─────────────────────────────────────────────────────────────────┐
│                        INSTITUTIONAL LAYER                       │
│  ┌─────────┐      ┌─────────┐      ┌─────────┐      ┌─────────┐ │
│  │   BTC   │ ───► │ SYGNUM  │ ───► │   HUB   │ ───► │  iBTCY  │ │
│  │         │      │ CUSTODY │      │         │      │         │ │
│  └─────────┘      └─────────┘      └─────────┘      └─────────┘ │
└───────────────────────────────────────┬─────────────────────────┘
                                        │
                                        ▼
┌─────────────────────────────────────────────────────────────────┐
│                          DeFi LAYER                              │
│  ┌─────────┐      ┌─────────┐      ┌─────────────────────────┐  │
│  │  iBTCY  │ ───► │  BTCY   │ ───► │   LENDING  │  DEX  │ ...│  │
│  │         │      │  VAULT  │      │   MARKETS  │ POOLS │    │  │
│  └─────────┘      └─────────┘      └─────────────────────────┘  │
└─────────────────────────────────────────────────────────────────┘
```

<Steps>
  <Step title="Mint iBTCY">
    Send BTC to Sygnum custody. Receive iBTCY at NAV price after KYC verification.
  </Step>
  <Step title="Wrap to BTCY">
    Deposit iBTCY into the vault. Receive BTCY 1:1. No fees, instant.
  </Step>
  <Step title="Use in DeFi">
    BTCY works like any ERC-20. Lend it, LP it, use as collateral.
  </Step>
  <Step title="Unwrap & Redeem">
    Convert BTCY → iBTCY → BTC when you're ready to exit.
  </Step>
</Steps>
