# Quick Start Guide

<Steps>
  <Step title="Get Allowlisted">
    Complete KYC verification to receive iBTCY.
    
    <Card title="Start KYC" icon="user-check" href="https://app.btcy.fi/onboard">
      Complete verification on the BTCY app
    </Card>
  </Step>
  
  <Step title="Mint iBTCY">
    Send BTC to custody and receive iBTCY at NAV price.
    
    - Minimum: $100,000 USD equivalent
    - Processing: T+0/T+1 for instant, monthly dealing for queued
    
    [Learn more about minting →](/ibtcy/minting)
  </Step>
  
  <Step title="Wrap to BTCY (Optional)">
    Deposit iBTCY into the vault to receive BTCY for DeFi use.
    
    ```solidity
    // 1. Approve
    ibtcy.approve(btcyVault, amount);
    
    // 2. Deposit
    btcy.deposit(amount, yourAddress);
    ```
    
    [Learn more about wrapping →](/btcy/wrapping)
  </Step>
</Steps>
