## Audit Scope

All audits cover:

- Access control logic
- Oracle integration
- Upgrade mechanisms
- Circuit breaker logic
- Force/recovery functions
- Reentrancy protection
- Integer overflow/underflow

## Bug Bounty Program

*Coming Soon*

We're launching a bug bounty program to reward security researchers who identify vulnerabilities.

### Severity Levels

| Severity | Example | Reward |
|----------|---------|--------|
| Critical | Fund theft, unauthorized minting | Up to $100,000 |
| High | Privilege escalation, oracle manipulation | Up to $25,000 |
| Medium | DoS, griefing attacks | Up to $5,000 |
| Low | Gas optimization, minor issues | Up to $1,000 |
